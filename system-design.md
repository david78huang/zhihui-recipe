## 功能模块

### 1. 每周自动生成新食谱 🔄

**触发条件：**
- 每周日晚上自动生成下周食谱
- 或手动触发

**功能：**
- 根据季节推荐时令菜
- 轮换蛋白质来源（鸡/牛/鱼/虾）
- 确保营养均衡
- 考虑 Jimmy 成长需求

**输出：**
- 更新 GitHub Pages 网页
- 生成 Obsidian 笔记

---

### 2. 食材智能推荐 📱

**触发条件：**
- 用户发送食材照片/清单
- 或输入现有食材

**功能：**
- AI 识别食材
- 推荐能做的菜品
- 考虑烹饪时间
- 营养搭配建议

**输出：**
- 当日推荐食谱
- 购物清单（缺什么）

---

### 3. 营养追踪分析 📊

**触发条件：**
- 每日记录实际饮食
- 每周生成报告

**记录内容：**
- 早餐/午餐/晚餐实际吃了什么
- Jimmy 的身高体重变化
- 特殊备注（不爱吃、特别爱吃等）

**分析输出：**
- 蛋白质/钙/维生素摄入统计
- 与推荐食谱对比
- 调整建议

## 技术实现

### 自动化工作流

```yaml
# .github/workflows/auto-recipe.yml
name: Auto Recipe Generator

on:
  schedule:
    - cron: '0 20 * * 0'  # 每周日晚8点
  workflow_dispatch:  # 手动触发

jobs:
  generate:
    runs-on: ubuntu-latest
    steps:
      - 生成下周食谱
      - 更新网页
      - 创建 Obsidian 笔记
      - 推送到 GitHub
```

### 用户交互方式

| 功能 | 触发方式 | 输出 |
|------|---------|------|
| 生成下周食谱 | 自动（周日）/ 说"生成下周食谱" | 网页更新 + Obsidian 笔记 |
| 食材推荐 | 发照片/文字"今天有XX食材" | 推荐菜品 + 购物清单 |
| 记录饮食 | 说"今天早餐吃了XX" | Obsidian 每日记录更新 |
| 查看报告 | 说"这周营养怎么样" | 营养分析报告 |

## 🥛 Jimmy青春期长高小贴士

14岁正是长高的黄金期，这些营养要跟上：

### 每天必吃
- ✅ **牛奶/酸奶** 500ml（早晚各一杯）
- ✅ **鸡蛋** 1-2个
- ✅ **瘦肉/鱼/虾** 适量

### 关键营养素
| 营养素 | 食物来源 |
|--------|----------|
| **钙** | 牛奶、豆腐、小鱼干、绿叶蔬菜 |
| **蛋白质** | 鸡蛋、瘦肉、鱼、豆制品 |
| **锌** | 牛肉、海鲜、坚果 |
| **维生素D** | 每天晒太阳20分钟 |

### 生活习惯
- 🌙 **晚上10点前睡觉** - 生长激素分泌高峰期
- 🏃 **适量运动** - 篮球、跳绳有助长高
- 🚫 **避免** - 碳酸饮料、过多油炸食品
